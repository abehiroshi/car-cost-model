# Welcab Assumptions

このメモは、Welcab固有装備の追加修理期待値を説明するための初期メモです。

## Added Repair Expected Value

- 福祉装備追加修理期待値: 年15,000円

## Target Equipment

対象として想定している装備は次の通りです。

- スロープ
- 車椅子固定装置
- 電動固定フック
- ワイヤー
- モーター
- センサー
- 専用ハーネス

## Usage Assumption

年間走行距離は2,500kmの低使用を想定しています。そのため、走行距離起因の劣化は一般的な使用より小さい一方、ゴム、樹脂、電装、機械部品の経年劣化は残るという考え方で、追加修理期待値を置いています。

## Update Policy

実費修理見積が出た場合は、期待値として薄くならすのではなく、該当年の実費イベントとして別途入力する方針です。v1ではその入力欄は未実装のため、必要になった時点で年次キャッシュフローへの個別イベント入力を追加します。
