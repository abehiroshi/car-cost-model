# knowledge

このディレクトリは、計算モデルの前提、数値根拠、未確認事項を記録する場所です。

ここにあるメモは最新Web情報ではありません。ChatGPTや人が、Web検索を使わずにリポジトリ内の情報だけで `assumptions.json` の数値前提を説明するための資料です。

v1では今日から明日の一次判断を優先し、初期メモの粒度に留めます。正式な見積、査定、ローン条件、修理見積が出た場合は、`docs/assumptions.json` と `inputs/assumptions.json` を更新し、対応する根拠または未確認事項をこのディレクトリへ追記します。

## Files

- `maintenance.md`: 車検、税金、保険、燃料、メンテ、修理期待値
- `welcab.md`: 福祉装備に関する追加修理期待値
- `depreciation.md`: 査定額、残価、価値低下の考え方
- `loan.md`: 現在ローンと新ローンの初期条件
