# Depreciation Assumptions

このメモは、査定額と残価の考え方を説明するための初期メモです。

## Current Appraisal

現在車両の査定額は3,014,000円を基準値としています。

現在車両の2026年時点の車齢は4年を初期値としています。車齢判定は `currentVehicle.ageAtStart` を優先し、`purchaseYear` は補助情報として残します。

## Value Model

年齢別価値テーブルは `assumptions.json` の `depreciation` を正とします。v1時点では詳細な市場価格テーブルをまだ持たず、簡易的な価値低下モデルで一次判断しています。

下取り額や最終車両価値は、年次キャッシュフローの年合計からは控除しません。車両価値と純資産として別表示し、家計支出と資産価値を混ぜない方針です。

車検前買替比較では、利用期間の最終年にその時点の車両価値を最終売却額として扱います。ただし、年次キャッシュフローの年合計には混ぜず、売却後実質コストの計算だけで控除します。

`最後まで維持・最終売却` は、評価期間中に買替を行わず、最後にその時点の現車価値で売却して終了する上限ケースです。

年間走行距離は2,500kmの低走行条件です。そのため、一般的な距離劣化よりも年式劣化を主軸にした簡易モデルとして扱います。

## Dealer Confirmation Items

次の点はディーラー確認事項です。

- 下取り保証期間
- 納車時査定との差額リスク
- 走行距離が少ないことによる査定上乗せ余地
- Welcab装備の中古市場評価

確認結果が出たら、査定額や下取り額を `assumptions.json` に反映し、このメモに根拠を追記します。
