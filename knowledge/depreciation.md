# Depreciation Assumptions

このメモは、査定額と残価の考え方を説明するための初期メモです。

## Current Appraisal

現在車両の査定額は3,014,000円を基準値としています。

現在車両の2026年時点の車齢は4年を初期値としています。車齢判定は `currentVehicle.ageAtStart` を優先し、`purchaseYear` は補助情報として残します。

## Value Model

車両価値は `assumptions.json` の `tradeInTable` を正とします。残価率による計算は廃止し、車齢ごとの査定額テーブルで下取り想定額、最終売却額、純資産の車両価値を計算します。

初期値は4年2,900,000円、6年2,100,000円、8年1,500,000円、10年1,000,000円、12年700,000円、14年400,000円です。中間年は隣接する2点の線形補間で扱います。たとえば5年は2,500,000円、7年は1,800,000円、9年は1,250,000円、11年は850,000円、13年は550,000円です。

14年以降は当面400,000円を下限値として扱います。これは今日から明日の一次判断用の仮置きであり、正式なディーラー査定や中古車相場を確認したら `tradeInTable` を更新します。

下取り額や最終車両価値は、年次キャッシュフローの年合計からは控除しません。車両価値と純資産として別表示し、家計支出と資産価値を混ぜない方針です。

買替時の下取り想定額は、`newVehicle.tradeIn` の固定値ではなく、その時点で手放す車両の `tradeInTable` 上の推定価値として表示します。これは資産移動の確認用であり、年合計には混ぜません。新車購入費用も支出合計とは別表示し、家計支出として年合計に入るのはローン支払いです。

車検前買替比較では、利用期間の最終年にその時点の車両価値を最終売却額として扱います。ただし、年次キャッシュフローの年合計には混ぜず、売却後実質コストの計算だけで控除します。

年間走行距離は2,500kmの低走行条件です。そのため、一般的な距離劣化よりも年式劣化を主軸にした簡易モデルとして扱います。

## Dealer Confirmation Items

次の点はディーラー確認事項です。

- 下取り保証期間
- 納車時査定との差額リスク
- 走行距離が少ないことによる査定上乗せ余地
- Welcab装備の中古市場評価

確認結果が出たら、査定額や下取り額を `assumptions.json` に反映し、このメモに根拠を追記します。
